.check-m

체크박스에 디자인을 입힌 형태로, input _ label을 <span class="wrap-check"> 로 감싸서 사용.
여러개의 체크박스가 필요한경우는 가로길이는 고정형(px), 유동형(%)로 나뉘어져 있으니 span에 추가하셔서 사이즈 조절 하시면 됩니다.
(ex. <span class="wrap-check wf150"> )
고정형은 (wf + 숫자) 형태이고, 유동형은 (w + 숫자) 형태입니다. 없는 사이즈는 추가 가능합니다.
